1) Uncompress the package so that the folders like bin and lib 
   are in c:\tmi2
2) Use wordpad to edit c:\tmi2\bin\config.tmi2.win32 and change the mudname
3) Use wordpad to edit c:\tmi2\lib\adm\etc\groups and change wherever it 
   says "cratylus" to whatever it is your admin user name will be.
4) Double-click on c:\tmi2\runmud.bat
5) telnet to localhost 5000

Or telnet to localhost on whatever port you specified in config.tmi2
to be your mud's listening port.

6) after you create your character, type: path -add /cmds/adm
7) quit
8) shut down the mud by closing its command window
9) start the mud again by double clicking on c:\tmi2\runmud.bat

To tune into the code discussion channel on intermud, once you log in, type:
tune code in

Enjoy!
